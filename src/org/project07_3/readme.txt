---------
Задание 3
---------

1. Клиент посылает через сервер сообщение другому клиенту.
2. Клиент при обращении к серверу методом getSonet() получает случайно выбранный сонет
   Шекспира из файла.
3. Сонеты должны храниться в базе данных

Реализация:
-----------
Задание реализовано путем использования двух потоков как в клиентской, так и в серверной частях.
Поскольку общение клиентов происходит через сервер, то был выбран протокол TCP для гарантированной доставки сообщений от клиента серверу.

Сначала должен запускаться сервер, который, используя TCP-сокет, принимает входящие соединения клиентов.
Вместе с тем, сервер запускает второй поток, который через протокол UDP транслирует входящие сообщения клиентам, которые, в свою очередь, фильтруют
их согласно разработанному протоколу.
Клиенты слушают UDP-группу, куда транслирует сервер.
Для того, чтобы отправить сообщение другому клиенту, необходимо отправить сообщение вида "to: имя_клиента текст сообщения".
Другой клиент, используя регулярные выражения, фильтрует сообщения, отправленные именно ему и выводит их на экран.
Для получения случайного сонета необходимо отправить сообщение "sonnet", для завершения работы - "bye".

Тексты сонетов находятся в папке sonnets проекта, пути к ним в базе данных sonnets (таблица sonnets, поле path). Выбираются случайным образом.
Для работы необходимо подключение библиотеки mysql-connector-java-5.1.41-bin.jar
